## How to Use

1. Open `index.html` in any modern web browser
2. Click **Start** to begin the stopwatch
3. Click **Stop** to pause
4. Click **Reset** to return to 00:00:00
5. Click **View Author** in the top-right corner for contact information

## Timer Behavior

- The hand completes one full rotation every 60 seconds
- Each rotation increments the minute counter
- Smooth continuous rotation without jumping
- Centisecond precision (1/100th of a second)

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and responsive design
- `samay.png` - Center image displayed in the clock
- `hand.png` - Clock hand image that rotates
- `clock-bg.png` - Background image for the clock face

## Customization

To personalize the stopwatch:

1. **Update Author Links** - Edit the social media URLs in `index.html`
2. **Change Images** - Replace `samay.png`, `hand.png`, or `clock-bg.png` with your own
3. **Modify Colors** - Update the color scheme in `styles.css`
4. **Adjust Clock Size** - Modify the `.clock` width/height values
